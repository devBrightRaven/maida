# Maida

**Steam 游戏库里 200 款。今晚玩什么毫无头绪。**

Maida 一次只展示一款游戏。玩或跳过。没有浏览、没有滚动、没有选择疲劳。

## 下载

**[下载 Maida](https://github.com/devBrightRaven/maida/releases/latest)** — Windows(.exe)跟 Linux(.deb、.AppImage)

![Rin 模式](screenshot/trimmed/rinView.png)

## 怎么用

1. Maida 读取你已安装的 Steam 游戏
2. 出现一款游戏 — **试试** 或 **先跳过**
3. 按试试游戏就启动。按先跳过就出现下一款
4. 就这样。没清单、没评分、没推荐

![Kamae 模式](screenshot/trimmed/kamaeView.png)

## 功能

- **两种模式** — Rin 快速决策,Kamae 把游戏分类到型(以心情分组的清单)
- **从你身上学** — 每次选择带权重,每天衰减回中性
- **4 种语言** — 英文、日文、繁体中文、简体中文
- **键盘跟手柄** — 十字键跟左摇杆操作,右摇杆滚动长页面,为 ROG Ally 等掌机设计
- **屏幕阅读器支持** — Windows NVDA 测试过
- **自动更新** — 未来版本自动安装

## 更多信息

- **使用手册**: [在线版(英文)](https://brightraven.world/maida/manual/) 或 [Gist 上的 4 语言版](https://gist.github.com/devBrightRaven/b26da3e8aef40d683a92e66e5b783fec)
- **隐私**: 游戏数据全部留在你的设备上;启动时一次匿名 ping,可在设置关闭
- **无障碍**: 键盘、屏幕阅读器(NVDA 测过、Orca 部分支持)、焦点管理、尊重 prefers-reduced-motion。完整声明见 [brightraven.world/accessibility/](https://brightraven.world/accessibility/)
- **开发**: 安装、测试、编译步骤见 [DEVELOPMENT.md](DEVELOPMENT.md)
